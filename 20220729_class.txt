# 지도학습
 - 회귀(Regression)
	- 예측하고자 하는 target값이 연속형
 
 
 - 분류(Classification)
	- 예측하고자 하는 target값이 번주형

# 지도학습 - 분류의 종류
	- 이진분류 : 범주형의 범주 개수가 2개
	- 다항분류 : 범주형의 범주 개수가 3개 이상

# 과적합 or 과대적합(overfitting)
	- 학습한 것에만 딱 막게 학습한 것.
	- 학습 데이터만 성능이 최적화되어서, 일반적인 데이터에는 잘 최적화되어 있지 않다.
	
# 과소적합(underfitting)
과대적합(overfitting)의 반대현상.

# knn 모델
	- [분류] 새로운 데이터와 기존데이터 거리 가까운 것, (k=3)이면 세게의 데이터의 
	- [회귀] 새로운 데이터와의 거리에 따라서 k가 3이라고 하면, 세게의 값의 target 평균으로 예측하는 모델

# 선형 회귀(Linear Regression)
 - 선형 함수로 표현이 된다.
 - 특징 하나 - 직선, 특징이 두개 - 평면, 특징이 세게 이상 - 초평면으로 표현된다.
 - 선형모델에서 MSE가 최소가되는 w, b가 정해진다.
	w : 회귀계수 , b : 편형
