하이퍼 파라미터 : 개발자, 사용자가 결정할 수 있는 파라미터
모델 파라미터 : 모델이 학습해서 결정하는 파라미터, 사람이 결정할 수 없음

* k의 갯수가 작으면 작을수록 과대적합이 될 가능성이 있다.
* K의 갯수가 크면 클수록 완만한 곡선(과소적합)

회귀 모델 
 - 예측하려는 값이 수치형일떄 사용 
 - MSE(Mean Squared Error)최소하 시키는 회귀계수를 찾는다.
   회귀계수(w1, w2,...b)
 - 선형 회귀(Linear Regression)
 - 선형함수로 표현되는데, w와 b가 모델 파라미터가 된다.
 - 특성이 하나일떄, 직선 표현, 특성이 두개일때,
   y = w1*x1 + b -> 직선
   y = w1*x1 + w2 * x2 + b -> 평면
 - 특성이 3개이상
   y = w1* x1 + w2 * x2 + w3 * x3 + b 초평면
